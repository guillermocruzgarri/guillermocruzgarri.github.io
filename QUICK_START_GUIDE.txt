🚀 QUICK START (5 MINUTES)
═══════════════════════════════════════════════════════════════════════════

1. EXTRACT THE ARCHIVE
   
   tar -xzf guillermo_cruz_portfolio_code.tar.gz
   cd portfolio_code

2. INSTALL DEPENDENCIES
   
   npm install

3. RUN LOCALLY
   
   npm run dev
   
   Open: http://localhost:3000

4. DEPLOY TO VERCEL (EASIEST)
   
   a. Push to GitHub:
      git init
      git add .
      git commit -m "Initial commit"
      git branch -M main
      git remote add origin https://github.com/guillermocruzgarri/guillermocruzgarri.github.io.git
      git push -u origin main
   
   b. Go to vercel.com → Import project → Deploy
   
   Done! Your site is live in 2 minutes!

5. ALTERNATIVE: DEPLOY TO GITHUB PAGES
   
   a. Update next.config.js (see INSTALLATION_GUIDE.md)
   b. npm install --save-dev gh-pages
   c. npm run deploy
   
   Site live at: https://guillermocruzgarri.github.io
