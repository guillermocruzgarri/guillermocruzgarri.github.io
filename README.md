
# Guillermo Cruz - Professional Portfolio

Portfolio website showcasing expertise in Business Analysis and Financial Data Analysis.

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- shadcn/ui

## Installation

```bash
yarn install
```

## Development

```bash
yarn dev
```

## Build

```bash
yarn build
```

## Deploy

This project is optimized for Vercel deployment.
